# 运维手册

## 1. 日常巡检
- 检查来源总览：`overview_metrics`
- 检查复制状态：`channel_status`
- 检查任务积压：`get_bootstrap_tasks`
- 检查错误日志：`get_source_logs` / `get_task_logs`

## 2. 常见故障处理

### 2.1 网络异常
- 现象：`test_source_connection` 失败
- 处理：检查主库端口、防火墙白名单、路由策略

### 2.2 GTID异常
- 现象：诊断提示 GTID 未开启或状态异常
- 处理：主从侧检查 `gtid_mode` / `enforce_gtid_consistency`

### 2.3 权限异常
- 现象：`access denied` 或复制线程失败
- 处理：核对复制账号权限、账号来源IP限制

### 2.4 任务卡死
- 处理：调用 `recover_bootstrap_tasks` 恢复滞留任务
- 若超重试上限，人工排查后重新创建任务

## 3. 建议告警项
- 失败任务数 > 0
- 复制错误来源数 > 0
- 初始化平均耗时持续升高

## 4. 变更管理
- 高危操作（删除来源、取消任务）必须二次确认
- 变更前导出配置快照，变更后验证关键链路
