## 阶段 2：真实 channel 操作接入（已完成）

### 阶段目标

将骨架状态切换升级为真实 MySQL 多源复制 SQL 操作，并补充基础诊断能力。

### 主要变更

- 后端能力升级：
  - `start_channel` 改为执行 `CHANGE MASTER TO ... FOR CHANNEL` + `START SLAVE FOR CHANNEL`
  - `stop_channel` 改为执行 `STOP SLAVE FOR CHANNEL`
  - `channel_status` 改为实时读取 MySQL 复制状态
  - 新增 `test_source_connection`（网络连通性检测）
  - 新增 `get_gtid_status`（GTID 模式检查）
- 前端能力升级：
  - 全部核心按钮与文案切换为中文
  - 新增 GTID 状态查询入口
  - 新增来源连通性检查
  - 新增按 source 的状态弹窗查看

### 自测结果

- `python -m py_compile mysql_multi_source_main.py` 通过
- IDE Lint 检查通过（无错误）
- API 与前端按钮映射关系检查通过
